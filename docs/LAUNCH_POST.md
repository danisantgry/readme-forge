I built readme-forge, a small TypeScript CLI for generating README drafts from real project metadata.

It works offline by default, supports optional Gemini refinement via env vars, and now has:

- npx usage
- --dry-run previews
- --check for missing README sections
- cli/library/web templates
- TypeScript, Vite, React, Next.js, Express, Python, Rust, and Go markers

I would love maintainer feedback: what README checks or ecosystem detectors would make this genuinely useful?
