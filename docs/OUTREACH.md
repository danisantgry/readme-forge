# Outreach Kit

Use these drafts to ask for real feedback. Do not spam communities, buy stars, or claim adoption that does not exist.

## Short Post

```text
I built readme-forge, a small TypeScript CLI that generates README drafts from real project metadata.

It works offline, supports --dry-run, --check, and --template cli|library|web, and can optionally use Gemini via environment variables.

You can try it without installing:

npx github:danisantgry/readme-forge . --dry-run

I am looking for maintainer feedback: what README checks or ecosystem detectors would make this genuinely useful?

https://github.com/danisantgry/readme-forge
```
