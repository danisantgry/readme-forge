# Contributing to readme-forge

Thanks for considering a contribution. `readme-forge` is a small maintainer-focused tool, so the best changes are practical, tested, and easy to review.

## Local Setup

```bash
npm install
npm run lint
npm run build
npm test
```

## Good First Contributions

- Add framework or package-manager detection.
- Improve README templates without inventing project facts.
- Add tests for real-world project shapes.
- Improve CLI error messages.
- Document examples from small open-source repositories.

## Pull Request Guidelines

- Keep changes focused on one behavior or documentation improvement.
- Add or update tests for parser, detector, or generator changes.
- Do not include API keys, generated secrets, or private repository data.
- Prefer deterministic output in offline mode.

## AI-Assisted Contributions

AI assistance is welcome when the final contribution is reviewed by a human. Generated README content should remain accurate, minimal, and grounded in repository metadata.
